# konastats.js
## available functions:
* `Stats.average(array)`
  * returns average of array
* `Stats.median(array)`
  * returns median of array
* `Stats.mode(array)`
  * returns value that occurs most
* `Stats.histogram(array)`
  * returns array of vectors, `[value, number of times it occurs]`
